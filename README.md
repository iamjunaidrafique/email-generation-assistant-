Email Generation Assistant

An AI-powered Email Generation Assistant that generates professional and structured emails based on user inputs such as intent, key facts, and tone.

🚀 Features

Generate professional emails automatically

Customize tone (formal, friendly, etc.)

Include key facts in the email

Structured and consistent output

Evaluation using custom metrics

🧠 Objective

To automate high-quality email drafting using prompt engineering techniques, improving efficiency and consistency.

⚙️ Prompting Technique

This project uses:

Role-Based Prompting → Guides the model to act as a professional email writer

Few-Shot Learning → Provides examples to improve output quality

📁 Project Structure
prompt_template.txt → Prompt used for generating emails

evaluation.json → Test scenarios

evaluation.csv → Evaluation results

metrics.json → Custom metric scores

report.pdf → Final analysis report

📌 Example

Input:

Intent: Job Application

Tone: Professional

Key Facts: Python Developer, 2 years experience

Generated Email:

Dear Hiring Manager,

I am writing to apply for the Python Developer position. I have 2 years of experience in software development and strong skills in Python.

I would welcome the opportunity to contribute to your organization.

Sincerely,
[Your Name]

📊 Custom Metrics

Fact Recall Score → Measures inclusion of key facts

Tone Accuracy Score → Measures tone correctness

Clarity Score → Measures readability and conciseness

📈 Results

The system generates clear, structured, and professional emails.
See evaluation.csv and report.pdf for detailed results.
